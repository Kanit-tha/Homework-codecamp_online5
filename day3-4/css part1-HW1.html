For example css syntax

css syntax 

           Selector {Declaration}
           Selector {property:value;}

         
example
  

        table,td,th  {
            border:1px solid black;
        }
        td,th {
            padding: 15px;
        }

        Selector:table,td,th
        หรือคือ <table>
              <td> 
              <th> 

       
        Declaration:ส่วนที่อยู่ใน {}
            Property:border และ padding
            value:1px, solid, black และ 15px

Selector example case


        <h1 id="nameId" class="nameClass">Selector example case</h1>
        
ID : 


        #nameId {
            background-color: rosybrown;
        }

        
Class :


        .nameClass {
            border:1px solid black;
        }

        


        
Pseudo element :


        h1 :: first-letter {
            color:red;
        }

        


        
Pseudo class:

        h1 : hover {
            color: blueviolet;
        }

        



        
Attribute Selector:


        
        <input type="button" value="button">

        input[type] {
            background-color: blueviolet;
        }
