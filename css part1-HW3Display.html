value
  • inline:
    อยู่บรรทัดเดียวกันกลับ element ก่อนหน้า
  • block:
    อยู่คนละบรรทัดกับ element ก่อนหน้า
  • contents:
    ทำให้กลายเป็น contents
  • flex:
    กำหนดจัดรูปแบบเป็น flex
  • grid:
    กำหนดจัดรูปแบบเป็น grid
  • inline-(block, flex, grid, table):
    เปลี่ยน inline ให้มีคุณสมบัติแบบอื่น
  • run-in:
    เปลี่ยนเป็น block or inline ขึ้นอยู่กับเนื้อหา
  • none:
    ซ้อนเนื้อหา
Different
    ขนาดการแสดงผล:
  • grid กำหนดขนาดจำนวนแน่ชัด
  • flex กำหนดคราวยืดหยุน ปรับเปลี่ยนได้
    รูปแบบ
  • grid เป็นตาราง row-column
  • flex เป็น การจัดเรียงตามแนวแกน row หรือ column
    item
  • grid ระยะ การรวมขยายช่อง
  • flex ลำดับ การยืดหดจำนวนเท่า
 Different
    แนวแกน
  • justify จัดในแนวแกน x หรือแนวนอน (ซ้าย กลาง ขวา)
  • align จัดในแนวแกน y หรือแนวตั้ง (บน กลาง ล่าง)
