For example css syntax

css syntax 

           Selector {Declaration}
           Selector {property:value;}

         
example
  

        table,td,th  {
            border:1px solid black;
        }
        td,th {
            padding: 15px;
        }

        Selector:table,td,th
        หรือคือ <table>
              <td> 
              <th> 

       
        Declaration:ส่วนที่อยู่ใน {}
            Property:border และ padding
            value:1px, solid, black และ 15px

        <h1 id="nameId" class="nameClass">Selector example case</h1>

        #nameId {

        }

        
Class :


        .nameClass {
            border:1px solid black;
        }

        


        
Pseudo element :


        h1 :: first-letter {
            color:red;
        }

        h1 : hover {
            color: blueviolet;
        }

Property Value Unit Differentiation
คือ คำที่ระบุองค์ประกอบ ส่วน พื้นที่ ตำแหน่ง ที่จะทำการกำหนดลักษณะ และคุณสมบัติ มี 5 ประเภท
  • Property for Content
  • Property for Border
  • Property for Padding
  • Property for Margin
  • Property for Element
 คือ คำที่ระบุหน่วย ตัวเลข คำวิเศษณ์ ระบุขนาดคุณสมบัตินั้นของpropert คำอันระบุเป็นลักษณ์
  • Value : คำบ่งบอกลักษณะ เช่น แดง ทึบ กลาง sans serif
  • Value : Unit ตัวเลขและหน่วยกำกับ
    • Absolute
    • Relative
 มี 2 ประเภท
  • Absolute Lengths
    • cm เซนติเมตร
    • mm มิลลิเมตร
    • in นิ้ว
    • px พิกเซล 1/96 นิ้ว
    • pt พอยท์ 1/72 นิ้ว
    • pc พิคาล 1/12 นิ้ว
  • Relative Lengths
    • em กี่เท่าของขนาดฟอนต์
    • ex กี่เท่าความสูงของฟอนต์
    • ch ความกว้างของตัวหนังสือใน parent
    • rem กี่เท่าขนาดฟอนต์ของroot
    • vh เปอร์เซ็นความสูงของหน้าจอ
    • vw เปอร์เซ็นความกว้างของหน้าจอ
    • vmin เปอร์เซ็นของ view point ต่ำสุด
    • vmax เปอร์เซ็นของ view point สูงสุด
    • % : 1-100 เปอร์เซ็นของ parent

Property & Value

                    Property: ระบุคุณสมบัติของส่วนประกอบนั้น
                    Value: ระบุขนาด หน่วย มีตัวเลขสำหรับระบุ คำบ่งบอกลักษณะ คำวิเศษณ์
                    เช่น

                        สี    :แดง

                    Relative เป็นการกำหนดอ้างอิง เปรียบเทียบ
