value
  • inline:
    อยู่บรรทัดเดียวกันกลับ element ก่อนหน้า
  • block:
    อยู่คนละบรรทัดกับ element ก่อนหน้า
  • contents:
    ทำให้กลายเป็น contents
  • flex:
    กำหนดจัดรูปแบบเป็น flex
  • grid:
    กำหนดจัดรูปแบบเป็น grid
  • inline-(block, flex, grid, table):
    เปลี่ยน inline ให้มีคุณสมบัติแบบอื่น
  • run-in:
    เปลี่ยนเป็น block or inline ขึ้นอยู่กับเนื้อหา
  • none:

Different
    ขนาดการแสดงผล:
  • grid กำหนดขนาดจำนวนแน่ชัด
  • flex กำหนดคราวยืดหยุน ปรับเปลี่ยนได้
    รูปแบบ
  • grid เป็นตาราง row-column
  • flex เป็น การจัดเรียงตามแนวแกน row หรือ column
    item
  • grid ระยะ การรวมขยายช่อง
  • flex ลำดับ การยืดหดจำนวนเท่า
 Different
    แนวแกน
  • justify จัดในแนวแกน x หรือแนวนอน (ซ้าย กลาง ขวา)
  • align จัดในแนวแกน y หรือแนวตั้ง (บน กลาง ล่าง)
Position & Z-index
Position Z-index
    คือ การกำหนดตำแหน่งที่จะแสดงผล
    เช่น
  • static: ค่าพื้นฐาน เลื่อนตามหน้าเว็บปกติ
  • fixed: กำหนดยึดอยู่กับที่ ไม่เลื่อนตามหน้าเว็บ
  • relative: เลื่อนออกจากตำแหน่งเดิมตามที่กำหนดค่า
  • alsolute: เลื่อนออกจากมุมบนซ้ายของหน้าจอ ตามกำหนดต่า
  • sticky: รวม relative และ fixed ไว้ด้วยกัน โดยการกำหนด
    ให้ relativeก่อน แล้วเพิ่มเติมว่าให้ไป fixed ที่top bottom
    คือ การกำหนดลำดับความลึก อยู่หน้าอยู่หลัง สิ่งใดทับอยุ่บนสิ่งใด
    ใช้งานโดย
  • ค่าพื้นฐาน ทุกชิ้นจะเท่ากับ z-index 0
  • ค่ามากกว่าจะอยู่หน้า เช่น z-index 1 อยู่หน้า 0
  • ค่าที่น้อยกว่าจะอยู่หลัง เช่น z-index -1 อยู่หลัง 1 และ 0
วิธีใช้ 
                    <style>
                        selector {
                            z-index: 1;
                        }
                    </style>
